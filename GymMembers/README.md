# About Dataset

This dataset provides a detailed overview of **gym members' exercise routines, physical attributes, and fitness metrics**. It contains **973 samples** of gym data, including key performance indicators such as **heart rate**, **calories burned**, and **workout duration**.

Each entry also includes demographic data and experience levels, allowing for a comprehensive analysis of **fitness patterns**, **athlete progression**, and **health trends**.

## Key Features

| Variable                          | Description                                                                |
| --------------------------------- | -------------------------------------------------------------------------- |
| **Age**                           | Age of the gym member.                                                     |
| **Gender**                        | Gender of the gym member, either Male or Female.                           |
| **Weight (kg)**                   | Member’s weight in kilograms.                                              |
| **Height (m)**                    | Member’s height in meters.                                                 |
| **Max_BPM**                       | Maximum heart rate, measured in beats per minute, during workout sessions. |
| **Avg_BPM**                       | Average heart rate during workout sessions.                                |
| **Resting_BPM**                   | Heart rate at rest before the workout.                                     |
| **Session_Duration (hours)**      | Duration of each workout session in hours.                                 |
| **Calories_Burned**               | Total calories burned during each workout session.                         |
| **Workout_Type**                  | Type of workout performed, such as Cardio, Strength, Yoga, or HIIT.        |
| **Fat_Percentage**                | Body fat percentage of the gym member.                                     |
| **Water_Intake (liters)**         | Daily water intake during workouts, measured in liters.                    |
| **Workout_Frequency (days/week)** | Number of workout sessions per week.                                       |
| **Experience_Level**              | Level of experience, from beginner `1` to expert `3`.                      |
| **BMI**                           | Body Mass Index, calculated using height and weight.                       |

## Dataset Usefulness

This dataset is ideal for:

* Data scientists
* Health researchers
* Fitness enthusiasts
* Sports analysts
* Students working on statistical or machine learning projects

It can be used to study **exercise habits**, model **fitness progression**, and analyze the relationship between **demographic**, **physical**, and **physiological variables**.

With its wide range of variables, this dataset offers valuable insights into how different factors affect:

* Workout intensity
* Physical endurance
* Calories burned
* Fitness level progression
