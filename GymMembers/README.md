# Acerca del Dataset

Este dataset proporciona una visión detallada de las **rutinas de ejercicio, atributos físicos y métricas de condición física** de miembros de un gimnasio. Contiene **973 registros** con información relacionada con el rendimiento físico, como **frecuencia cardíaca**, **calorías quemadas** y **duración del entrenamiento**.

Cada registro también incluye datos demográficos y niveles de experiencia, lo que permite realizar un análisis completo sobre **patrones de entrenamiento**, **progresión física** y **tendencias de salud**.

## Características principales

| Variable                          | Descripción                                                                                      |
| --------------------------------- | ------------------------------------------------------------------------------------------------ |
| **Age**                           | Edad del miembro del gimnasio.                                                                   |
| **Gender**                        | Género del miembro del gimnasio, ya sea masculino o femenino.                                    |
| **Weight (kg)**                   | Peso del miembro en kilogramos.                                                                  |
| **Height (m)**                    | Estatura del miembro en metros.                                                                  |
| **Max_BPM**                       | Frecuencia cardíaca máxima, medida en latidos por minuto, durante las sesiones de entrenamiento. |
| **Avg_BPM**                       | Frecuencia cardíaca promedio durante las sesiones de entrenamiento.                              |
| **Resting_BPM**                   | Frecuencia cardíaca en reposo antes del entrenamiento.                                           |
| **Session_Duration (hours)**      | Duración de cada sesión de entrenamiento en horas.                                               |
| **Calories_Burned**               | Total de calorías quemadas durante cada sesión de entrenamiento.                                 |
| **Workout_Type**                  | Tipo de entrenamiento realizado, como Cardio, Fuerza, Yoga o HIIT.                               |
| **Fat_Percentage**                | Porcentaje de grasa corporal del miembro.                                                        |
| **Water_Intake (liters)**         | Consumo diario de agua durante los entrenamientos, medido en litros.                             |
| **Workout_Frequency (days/week)** | Número de sesiones de entrenamiento por semana.                                                  |
| **Experience_Level**              | Nivel de experiencia, desde principiante `1` hasta experto `3`.                                  |
| **BMI**                           | Índice de Masa Corporal, calculado a partir de la estatura y el peso.                            |

## Utilidad del dataset

Este dataset es ideal para:

* Científicos de datos
* Investigadores del área de salud
* Entusiastas del fitness
* Analistas deportivos
* Estudiantes que trabajan en proyectos estadísticos o de machine learning

Puede utilizarse para estudiar **hábitos de ejercicio**, modelar la **progresión física** y analizar la relación entre variables **demográficas**, **físicas** y **fisiológicas**.

## Potencial de análisis

Gracias a la variedad de variables que contiene, este dataset permite obtener información valiosa sobre cómo distintos factores influyen en:

* La intensidad del entrenamiento
* La resistencia física
* Las calorías quemadas
* La progresión del nivel de condición física
* La composición corporal
* Las tendencias generales de salud
